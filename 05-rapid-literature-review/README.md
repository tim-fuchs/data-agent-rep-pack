# Rapid, Multivocal Literature Review

## Research Question for the Review

What are requirements for an agentic AI system that supports scientists in creating source code for data analysis?

## Process

1. Article search:
   - See folder `01-article-search` and sub-folders.
   - We structured the search process steps based on the [PRISMA (Preferred Reporting Items for Systematic reviews and Meta-Analyses) guidelines](https://www.prisma-statement.org):
     1. [Article identification](./01-article-search/01-identification/README.md)
     2. [Article screening](./01-article-search/02-screening/README.md)
     3. [Article inclusion](./01-article-search/03-inclusion/README.md)
   - Find the (intermediate) results of the process in [search-process-overview.xlsx](./01-article-search/search-process-overview.xlsx) and in the README files of each sub-folder.
2. Thematic analysis
   - See folder [02-thematic-analysis](./02-thematic-analysis/README.md) folder.
   - We conducted a semi-automatic process with the help of LLMs.
