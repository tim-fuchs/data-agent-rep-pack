# Rapid Literature Review

## Research Question for the Review

What are requirements for an agentic AI system that supports data scientists and natural scientists in writing source code for data analysis?

## Process

1. Article search:
   - See `01-identification` folder and sub-folders.
   - We structured the search process steps based on the [PRISMA (Preferred Reporting Items for Systematic reviews and Meta-Analyses) guidelines](https://www.prisma-statement.org):
     1. Article identification
     2. Article screening
     3. Article inclusion
   - Find the (intermediate) results of the process in `01-identification/search-process-overview.xlsx` and in the README files of each sub-folder.
2. Thematic analysis
   - See `02-thematic-analysis` folder.
   - We conducted a semi-automatic process with the help of LLMs.
   - Find the results in the README file of the folder.
